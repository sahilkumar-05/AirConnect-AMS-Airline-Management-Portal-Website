# ✈️ AirConnect - Airline Management Portal 

A full-stack **Airline Management Portal** designed to manage airline operations efficiently. The project includes flight scheduling, passenger bookings, ticket generation, staff management, payment processing, and database management using PostgreSQL.

## 📌 Overview

This Airline Management System provides a centralized platform to manage airline operations including airlines, airports, aircrafts, flights, schedules, passengers, staff, bookings, ticket generation, and payment processing.

The system is built for learning, academic projects, and database management practice while demonstrating full-stack web development concepts.

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/AirConnect-AMS-Airline-Management-Portal-Website.git
```

2. Navigate to the project

```bash
cd AirConnect-AMS-Airline-Management-Portal-Website
```

3. Install dependencies

```bash
npm install
```

4. Configure PostgreSQL database.

5. Run the provided SQL script to create all tables and sample data.

6. Start the server

```bash
npm start
```
